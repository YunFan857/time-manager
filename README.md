# 时间管理

一个极简的时间管理网站，帮助用户规划每日行程，让时间安排连贯、有意义。

## 功能

- ✅ 添加日程
- ✅ 查看今日安排
- ✅ 今日意图设置
- ✅ 日程分类（工作/生活/学习/其他）

## 技术栈

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase（待集成）

## 开始使用

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

打开 [http://localhost:3000](http://localhost:3000) 查看。
